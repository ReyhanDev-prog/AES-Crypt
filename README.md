# AES-128 Simulator

Aplikasi ini digunakan untuk mencoba proses enkripsi dan dekripsi AES-128. Selain menampilkan hasil akhirnya, aplikasi ini juga memperlihatkan setiap langkah proses AES agar lebih mudah dipelajari.

## Isi Project

```
AES-Crypt/
├── index.html      # Aplikasi utama
├── package.json
├── vercel.json
└── README.md
```

## Cara Menjalankan di Komputer

1. Download atau clone project ini.
2. Buka file **index.html** menggunakan browser seperti Chrome, Firefox, atau Edge.
3. Aplikasi siap digunakan.

Tidak perlu menginstal program tambahan.

---

# Cara Mengunggah ke Vercel

## Cara 1 (Paling Mudah)

1. Upload project ke GitHub.
2. Masuk ke https://vercel.com
3. Login menggunakan akun GitHub.
4. Klik **Add New → Project**.
5. Pilih repository **AES-Crypt**.
6. Klik **Deploy**.
7. Tunggu hingga selesai.

Setelah selesai, Vercel akan memberikan link website yang bisa langsung dibuka.

---

## Jika Deploy Gagal

Pastikan:

- File **index.html** berada di folder utama project.
- Tidak mengisi **Build Command**.
- Tidak mengisi **Output Directory**.
- Framework dipilih **Other** atau **No Framework**.

---

## Catatan

- Aplikasi ini berjalan sepenuhnya di browser.
- Tidak memerlukan database.
- Tidak memerlukan server.
- Tidak memerlukan API.
